Interactive Complete GraphSource Code
Video

This is something I felt like making one night. Two coding sessions later and this came out. You place nodes on the screen, and they connect themselves to every other node on the screen. Nodes will disappear when they collide with each other. The shortest edge is highlighted green, and the longest red. It makes for some cool looking images.

Controls:

  • H: help on/off
  • Left mouse: place node
  • Scroll wheel: steer nodes
  • Left and Right arrow keys: decrease/increase node velocity
  • Up and Down arrow keys: decrease/decrease blur
  • F: toggle fullscreen

To run this program you can either:

  • 1. Just run the SFML_graphTest.exe under the Release folder
  • 2. Compile it yourself. You'll need to change the 'additional includes' and 'additional library directories' to point to your local SFML location.

Built with SFML 2.1 and VS2012.

Data LoggerDownload
Source code

I created this program with a friend who wanted to set up an Arduino with some sensors (g force, temperatures, etc.) and plot the data. This is the result of just playing around with the idea.This program takes in a text file containing data (taken from an Arduino) and plots all the data. It computes the min, max, and average of each data set as well.The program is also capable of reading data realtime from an Arduino given that the Arduino is just serial printing a series of numbers in the correct format. Choose the COM port your Arduino is on from the Serial menu.

This program was made specifically for the Arduino set up we have. It isn't too flexible (try feeding a file with a different amount of columns), so that's why it's in the Experiments section.

Programmed in C# with WinForms for the GUI. The .zip download contains a sample data file.
